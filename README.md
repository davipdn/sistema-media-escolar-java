# Sistema de Cálculo de Nota do Aluno

Um mini projeto em Java para calcular a média das notas de um aluno com sistema básico de autenticação.

## 📋 Descrição

Este projeto consiste em uma aplicação Java simples que permite a um usuário fazer login e calcular a média de suas notas escolares. O sistema determina automaticamente o status do aluno (aprovado, em recuperação ou reprovado) com base na média calculada.

## 🚀 Funcionalidades

- **Sistema de Login**: Autenticação básica com usuário e senha
- **Entrada de Notas**: Permite inserir 4 notas diferentes
- **Cálculo Automático**: Calcula a média das notas inseridas
- **Classificação por Status**: 
  - **Reprovado**: média ≤ 2.9
  - **Recuperação**: média entre 3.0 e 5.9
  - **Aprovado**: média ≥ 6.0

## ▶️ Como Executar

1. Certifique-se de ter o Java 21 instalado em sua máquina
2. Compile o arquivo Java:
   ```bash
   javac src/applications/NotaAluno.java
   ```
3. Execute a aplicação:
   ```bash
   java applications.NotaAluno
   ```
4. Faça login usando as credenciais fornecidas
5. Digite as 4 notas quando solicitado
6. Visualize sua média e status final
